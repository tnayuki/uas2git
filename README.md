# uas2git

A tool for migrating Unity Asset Server projects to git

## Installation

    gem install uas2git

## Usage

    Usage: uas2git PROJECT_NAME [options]

    Specific options:
        -H, --host NAME                  Unity Asset Server host
        -u, --username NAME              Crendential for Unity Asset Server
        -p, --password PASSWD

        -h, --help                       Show this message
